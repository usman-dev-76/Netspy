# NetSpy Project

NetSpy is a Python-based networking tool developed for learning and practicing core networking concepts.

## Features:
- Network communication handling
- Data collection and processing
- Automation of basic network tasks
- Simple command-line based execution

## Technologies Used:
- Python
- Socket Programming
- Networking Fundamentals

## Purpose:
This project is built for educational purposes to understand how networking works using Python.

## How to Run:
1. Open terminal
2. Navigate to project folder
3. Run the script:

python main.py
